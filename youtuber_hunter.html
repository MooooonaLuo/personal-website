YouTuber Hunter

UIUX & Web Development · 2019

YouTuber Hunter is a website for companies or organizations to seek YouTubers to promote their products on YouTube channel. Users are able to search YouTubers by their names, channel categories and number of followers. It is made with programming languages HTML, C++ and Javascript. We used firebase to store user data and SQLite3 to get YouTuber data from our database. This project focuses more on coding than UI/UX.

Background & Concept

The requirement of this project is to design a product for people different from us, meaning none of our team members are target users. Traditional advertising agents are gradually being replaced by internet celebrities. Therefore, we came up with an idea of creating a bridge between YouTubers and companies who need product promotion. The website allows users to search YouTubers by channel names, categories and numbers of followers, and provides users with YouTubers' basic information.

Coding Up

We used firebase to store user account information and their YouTuber collection. We use SQLite3 as a tool to store and get the data about YouTubers from the database. For data display, we use ajax to get data from back end based on user inputs, including YouTuber categories, YouTuber names, and the number of followers.
